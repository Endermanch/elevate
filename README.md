### Usage
Prepend **quiet** to your command line to run the target silently.
The command line switches are case-insensitive.

`QUIET [/E] C:\Windows\Application.exe [ARGS]`

| Switch | Meaning                      |
|--------|------------------------------|
| /E     | Run with elevated privileges |
